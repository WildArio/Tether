# ⚡ Tether

**Tether** — инженерный калькулятор для проектировщиков электрики, АСУ ТП и систем ТГВ. Программа помогает выполнять расчёты и подбирать электрооборудование:

- 🔹 Расчёт номинального тока по мощности, напряжению, cosφ и КПД.  
- 🔹 Подбор автоматических выключателей (характеристики B, C, D).  
- 🔹 Выбор сечения кабеля по допустимому току (ПУЭ) и проверка падения напряжения.  
- 🔹 Подбор электродвигателя по ближайшей мощности или току.  

Все физические величины хранятся и вычисляются в **Международной системе единиц (СИ)** — ватты, амперы, вольты, омы на метр, эффективность в долях единицы.

---

## 📌 Текущее состояние

Проект находится в активной разработке.  
**Реализовано:**

- ✅ Базовые структуры данных (`Breaker`, `Cable`, `Motor`).  
- ✅ Статические каталоги оборудования (медные кабели ПУЭ, стандартный ряд автоматов, модельный ряд двигателей).  
- ✅ Функции загрузки каталогов.  

**В планах:**

- ⏳ Функции расчёта тока и падения напряжения.  
- ⏳ Алгоритмы подбора автомата и кабеля по расчётным параметрам.  
- ⏳ Поиск ближайшего электродвигателя.  
- ⏳ Возможность загрузки каталогов из внешних файлов (JSON).

## 🛠️ Сборка

Проект написан на **C++17**. Для сборки требуется компилятор с поддержкой C++17.

1. Склонируйте репозиторий.  
2. Откройте решение в Visual Studio (или соберите через CMake).  
3. Убедитесь, что корневая папка `Tether/src` добавлена в пути включения заголовков.  
4. Скомпилируйте и запустите.

## 🧪 Использование

Пример (после реализации функций подбора):

```cpp
#include "tether.h"

int main() {
    auto breakers = Tether::Data::loadBreakerCatalog();
    Tether::Calc::BreakerSelectionContext ctx{32.5, true}; // 32.5 A, motor load
    auto breaker = Tether::Calc::selectBreaker(breakers, ctx);
    if (breaker) {
        // выбран автомат
    }
    return 0;
}
```
